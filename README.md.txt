 Simple Calculator in Python

  Overview
This is a basic command-line calculator built using Python.  
It allows the user to perform Addition, Subtraction, Multiplication, and Division on two numbers.

The program:
1. Displays a menu of operations.
2. Takes the user's choice.
3. Accepts two numbers.
4. Performs the chosen mathematical operation.
5. Displays the result.



 Features
 Supports Addition, Subtraction, Multiplication, and Division.
 Handles division by zero errors gracefully.
 Accepts both integer and decimal numbers.
 Simple and beginner-friendly code structure.
